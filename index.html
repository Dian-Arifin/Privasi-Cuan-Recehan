Kebijakan Privasi

Cuan Recehan

Kebijakan Privasi ini menjelaskan bagaimana Cuan Recehan mengumpulkan, menggunakan, menyimpan, dan melindungi data pengguna saat menggunakan layanan kami.

1. Informasi yang Kami Kumpulkan

  • Nama pengguna (username)
  • Alamat email Gmail
  • Informasi perangkat dan browser
  • Alamat IP dan aktivitas login
  • Data transaksi dan riwayat misi
  • Data referral dan aktivitas withdraw

2. Penggunaan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Memproses registrasi dan login pengguna
  • Menjalankan sistem reward dan withdraw
  • Mencegah penyalahgunaan dan kecurangan
  • Meningkatkan keamanan platform
  • Menyediakan layanan dan dukungan pengguna

3. Perlindungan Data

  • Kami berusaha menjaga keamanan data pengguna.
  • Akses terhadap data dibatasi hanya untuk keperluan sistem.
  • Sistem keamanan digunakan untuk mendeteksi aktivitas mencurigakan.
  • Kami tidak menjual data pribadi pengguna kepada pihak lain.

4. Cookies dan Teknologi Pelacakan

Website dapat menggunakan cookies, session storage, local storage, dan teknologi serupa untuk menjaga keamanan akun, menyimpan sesi login, dan meningkatkan pengalaman pengguna.

5. Keamanan Sistem

  • Sistem dapat mendeteksi VPN, Proxy, dan aktivitas mencurigakan.
  • Perangkat pengguna dapat dicatat untuk mencegah multi akun.
  • Aktivitas yang melanggar aturan dapat menyebabkan pembatasan akun.

6. Penyimpanan Data

Data pengguna dapat disimpan selama akun masih aktif atau selama diperlukan untuk kebutuhan keamanan, audit, dan operasional sistem.

7. Tautan ke Situs Pihak Ketiga

Beberapa misi atau iklan dapat mengarahkan pengguna ke situs pihak ketiga. Kami tidak bertanggung jawab atas isi maupun kebijakan privasi situs tersebut.

8. Hak Pengguna

  • Pengguna dapat meminta penghapusan akun.
  • Pengguna dapat menghubungi admin terkait data akun.
  • Pengguna wajib menjaga kerahasiaan akun masing-masing.

9. Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu tanpa pemberitahuan sebelumnya. Pengguna disarankan untuk memeriksa halaman ini secara berkala.

10. Persetujuan Pengguna

Dengan menggunakan layanan Cuan Recehan, pengguna dianggap telah membaca dan menyetujui seluruh isi Kebijakan Privasi ini.
